# LLM Fundamentals and Engineering

This repository holds a self-paced course on **LLM fundamentals and engineering**, written for **complete beginners**. You do not need a background in programming or machine learning to start reading.

## What this is

The main material is a single guided lesson that turns short classroom notes into full explanations, simple diagrams, and step-by-step ideas. The goal is **strong mental pictures**: what a large language model is, how the common “kinds” of models differ, what words like **token**, **context window**, and **transformer** mean in everyday language, and how a **chat** can feel like a memory when the system does not really store your life story.

## Who it is for

- People who are new to **LLMs** and want a calm first pass.  
- Learners who prefer **plain English** and small steps over jargon.  
- Anyone who will later add **code**; the main document explains tools only where they help, and you can read everything without installing software.

## Optional setup (for hands-on parts later)

Some lessons may mention **local models** (for example Ollama), a **Python** environment, or a **code editor** (for example Cursor). You can still **read** the full course with **no** installs. When you are ready to run examples, set an API key safely (for example the `OPENAI_API_KEY` environment variable) and never commit secrets to the repository.
